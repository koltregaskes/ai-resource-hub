# Contributing to AI Resource Hub

AI Resource Hub is launching in a deliberately strict mode.

The repo is public because the site is public, but community contribution is staged carefully.

Current policy:

- issues and pull requests may be disabled or tightly limited during launch
- unsolicited design, UI, workflow, scraper, database, or automation changes are out of scope
- the only future public intake categories are:
  - model updates
  - stats or figure corrections
  - content corrections

Important:

- this is not a general-purpose frontend contribution repo
- maintainers may close or decline items that do not match the current scope
- maintainers may convert a valid report into internal work instead of inviting a pull request
- public pull requests should not be opened unless a maintainer has explicitly confirmed the path

If public intake is enabled, please provide:

- a clear summary of the problem
- the exact affected page, section, or claim
- source links or evidence for factual corrections
- the smallest possible correction scope

For pull requests:

- link the approved issue
- keep the change tightly scoped
- do not touch generated outputs, workflows, or frontend implementation unless explicitly invited

Final approval and merge always remain with the maintainer.
